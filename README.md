# Анализ результатов регионального этапа Всероссийской олимпиады школьников в Красноярском крае (2022-2025)
Анализ влияния перехода на московское время на результаты регионального этапа Всероссийской олимпиады школьников в Красноярском крае, а также анализ общих региональных тенденций по 4 предметам (литература, математика, биология, обществознание)

## Синопсис
В 2024 году произошло важное изменение в проведении регионального этапа Всероссийской олимпиады школьников: время начала олимпиадных состязаний привели в соответствие с московским. Участники олимпиады в Красноярском крае с 2024 года пишут олимпиаду с 13:00 вместо 9:00. В этом исследовании анализируется, как это изменение повлияло на результаты участников, а также изучаются общие тенденции: распределение прошедших на заключительный этап по предметам и динамика популярности олимпиад (количество участников). Использованы данные по четырём предметам (математика, биология, литература, обществознание) за 2022–2025 годы. ## Актуальность
Изменение времени начала интеллектуальных соревнований может влиять на когнитивные способности участников из-за сдвига циркадных ритмов, усталости или изменения режима дня. Понимание такого влияния важно для оптимизации расписания олимпиад и повышения объективности результатов. Кроме того, анализ популярности предметов и распределения прошедших на всероссийский этап даёт представление о развитии олимпиадного движения в регионе и может показать возможные векторы развития олимпиадной среды.

## Данные
Источниками данных послужили сайты Министерства просвещения Российской Федерации (файлы с данными о проходных баллах https://docs.edu.gov.ru/#activity=) и сайт Министерства образования Красноярского края (https://krao.ru/dokumentyi/drugie/vserossijskaya-olimpiada-shkolnikov/regionalnyij-etap-vserossijskoj-olimpiadyi-shkolnikov/arhiv-dokumentov-regionalnogo-etapa-vserossijskoj-olimpiadyi-shkolnikov/). Данные были собраны в виде pdf-файлов, а затем обработаны с помощью сайта https://www.ilovepdf.com/ru, анонимизированы и превращены в 16 файлов формата Excel. Все 16 таблиц были объединены в одну с помощью скрипта, написанного на Python с библиотеками pandas и os. Из 4 pdf-файлов была создана таблица проходных баллов, с помощью pd.merge() по трём ключам (предмет, год, класс) пороги присоединены к основному dataframe. Была проведена нормализация баллов для последующего более наглядного сравнения: добавлен столбец "Нормализованные баллы" по формуле (Итог / max_score) * 100, где max_score — максимальный возможный балл для данного предмета и года (взят из словаря, например, для математики 70, для биологии до 2024 - 100, с 2024 уже 312). Файл с очищенными данными, расчетами и выводами находится в папке [data](https://github.com/tumanovavioletta086-stack/regional-olympiad-time-impact-and-representation/tree/c8073ef87bffbb08ec2dd79d826ae017785845fe/data).

## Анализ
С помощью формулы МЕДИАНА(ЕСЛИ()) посчитаны медианные баллы для каждого предмета с 9 по 11 класс с 2022 по 2025 год и построены линейные диаграммы. Вместе с этим была создана сводная таблица, показывающая доли прошедших на всероссийский этап от общего количества, на ее основе построена линейная диаграмма. Ключевой вывод - перевод времени если и повлиял, то не так сильно, большее значение сыграли другие факторы.
Построена сводная таблица, отображающая общее количество участников по каждому предмету за каждый год, на ее основе построена столбчатая диаграмма. Основной вывод - олимпиада по математике самая массовая за 4 года, при этом в количестве ее участников произошел резкий скачок в 2023 году. Также популярность набирает олимпиада по обществознанию. ВСОШ в регионе становится все более массовым с каждым годом.
Была создана сводная таблица, показывающая общее количество участников всероссийского этапа ВСОШ по предметам. Таблица послужила основой для круговой диаграммы, которая показала, что на заключительном этапе олимпиады Красноярский край часто представляют не только математики, коих действительно большинство, но и участники олимпиады по литературе. Однако за 2022-2025 года Красноярский край совсем не был представлен на олимпиаде по обществознанию.
